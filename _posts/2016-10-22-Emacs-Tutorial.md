## 2. Basic Commands

* `C` -> "Ctrl"
* `M` -> "Alt"
* `C-h t` emacs tutorial
* `C-h v` display document of variable
* `C-h f` display document of function
* `C-h k` display document of function invoked by key
* `C-u number ***` execute *** number times
